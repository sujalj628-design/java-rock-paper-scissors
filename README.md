# Rock Paper Scissors Game

This is a simple Rock Paper Scissors game made using Java. It is a console-based application where the user plays against the computer. The computer randomly selects Rock, Paper, or Scissor, and the winner is decided based on the game rules.

## Features

- User vs Computer
- Random computer choice
- Score tracking
- Draw detection
- Input validation
- Quit the game anytime by typing `quit`

## Technologies Used

- Java
- IntelliJ IDEA
- Git & GitHub

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/sujalj628-design/java-rock-paper-scissors.git
```

2. Open the project in IntelliJ IDEA or any Java IDE.

3. Run `Main.java`.
